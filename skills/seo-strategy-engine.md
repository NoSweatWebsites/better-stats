# SEO Strategy Engine

## Purpose

Generate a commercially useful organic growth strategy by combining:
1. Strategic keyword ideation
2. Google Ads keyword data
3. Live SERP analysis
4. Competitor content analysis
5. Keyword clustering
6. Prioritisation
7. Content planning
8. Branded PDF report output

## Required Inputs

Ask for:
- Niche / industry
- Target audience
- Business goal
- Existing website URL, if available
- Target country / market
- Stage of site: new, growing, established
- Brand / offer context
- Whether the report is for internal use or client delivery

If inputs are missing, infer reasonable assumptions and state them clearly.

## Workflow

### Phase 1: Strategic Seeding

Generate:
- One primary keyword
- Three intent-separated subtopics
- Nine supporting content ideas using the 6 Circles method

Use the structure:
- Primary topic
- Subtopic A: problem-aware
- Subtopic B: solution-aware
- Subtopic C: comparison / buying-aware

Reject overlapping clusters.

### Phase 2: Keyword Expansion

Use keyword data from the app tools:
- Google Ads keyword ideas
- Search volume
- competition index
- CPC where available
- related terms

Expand to 50–100 keywords.

### Phase 3: SERP Reality Check

For priority keywords, analyse the top-ranking results.

Assess:
- content type
- search intent
- content depth
- page format
- freshness
- domain authority proxy
- weak pages
- missing sections
- AI-content vulnerability
- whether a new page can realistically compete

### Phase 4: Scoring

Score every keyword:

- Demand: 1–10
- Organic difficulty: 1–10
- Intent fit: 1–10
- Business value: 1–10
- SERP weakness: 1–10
- Opportunity score = Demand + Intent Fit + Business Value + SERP Weakness - Organic Difficulty

Prioritise:
- High-conviction bets
- Quick wins
- Supporting content
- Ignore / kill list

### Phase 5: Strategy Output

Generate:
- Executive summary
- Keyword cluster analysis
- Priority keyword table
- Quick wins
- Content pillars
- Page slug recommendations
- SERP gaps
- AI competition audit
- Internal linking plan
- Social distribution recommendations
- 14-day sprint
- 30/60/90-day execution timeline

### Phase 6: Report Rules

Be commercially decisive.
Do not produce a generic keyword dump.
Always explain why a keyword matters.
Always include what not to pursue.
Prioritise revenue and conversion potential over traffic alone.
